PipesGen

Produces patterns based on variations of the included pipes SynthDef.

Basic Usage:

Generate (and store) a pipes variation: 

PipesGen.new.genStorePipeSynth(intensity)

Generate a Pattern: 

PipesGen.new.genPattern(intensities)

Instance Methods

  • pipesGen(index = 0,count = 1,n = 5,intervals = [0,3,5,7,10,12,15,17,19]) Returns (but does not store) a SynthDef based on pipes. index is an integer to be appended to the SynthDef name to avoid collisions. count is the number of Klanks to mix together. n is the number of oscillators per Klank. intervals is the scale degrees of the base frequency of each successive Klank.
  • randIntervals(length = 20) Returns a random array of intervals.
  • genPipeSynth(intensity = nil,intervals = nil) Returns a pipe synth generated semi-randomly based on intensity and intervals. intervals and intensity are set randomly if unspecified.
  • genStorePipeSynth(intensity = nil,intervals = nil) Calls genPipeSynth and stores (and returns) the resulting synth.
  • genPattern(intensities,length,voice) Generates and returns a Pattern with a random synth based on intensities that lasts for length steps. voice is added as a key to the Pattern for debugging purposes.
